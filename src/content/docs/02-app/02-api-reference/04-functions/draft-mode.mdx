`draftMode`功能允许您检测[服务器组件](/docs/app/building-your-application/configuring/draft-mode)内部的[草稿模式](/docs/app/building-your-application/rendering/server-components).

```jsx filename="app/page.js"
import { draftMode } from "next/headers";

export default function Page() {
  const { isEnabled } = draftMode();
  return (
    <main>
      <h1>My Blog Post</h1>
      <p>Draft Mode is currently {isEnabled ? "Enabled" : "Disabled"}</p>
    </main>
  );
}
```
