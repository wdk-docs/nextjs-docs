在开发过程中，默认情况下会启用源映射。在生产构建过程中，它们被禁用，以防止您在客户端上泄露源代码，除非您特别选择使用配置标志。

Next.js 提供了一个配置标志，可用于在生产构建期间启用浏览器源映射生成：

```js filename="next.config.js"
module.exports = {
  productionBrowserSourceMaps: true,
};
```

当`productionBrowserSourceMaps`选项被启用时，源映射将在与 JavaScript 文件相同的目录中输出。Next.js 将在收到请求时自动提供这些文件。

- 添加源映射可以增加`next build`时间
- 在`next build`期间增加内存使用量
