Next.js 默认情况下将为每个页面生成[etags](https://en.wikipedia.org/wiki/HTTP_ETag)。您可能需要根据缓存策略禁用 HTML 页面的 etag 生成。

打开`next.config.js`并禁用`generateEtags`选项：

```js filename="next.config.js"
module.exports = {
  generateEtags: false,
};
```
