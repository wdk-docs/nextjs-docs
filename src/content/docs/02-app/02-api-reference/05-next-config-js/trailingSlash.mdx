默认情况下，Next.js 会将带有尾部斜杠的 URL 重定向到没有尾部斜杠的对应 URL。例如，`/about/`将重定向到`/about`。 您可以将此行为配置为相反的方式，即没有尾随斜杠的 url 将重定向到带有尾随斜杠的对应 url。

打开`next.config.js`并添加`trailingSlash`配置：

```js filename="next.config.js"
module.exports = {
  trailingSlash: true,
};
```

设置此选项后，像`/about`这样的 URL 将重定向到`/about/`。
