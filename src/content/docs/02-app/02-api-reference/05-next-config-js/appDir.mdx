> **很高兴知道**: 从 Next.js 13.4 起，**不再**需要此选项。应用程序路由器现已稳定。

应用程序路由器 ([`app` 目录](/docs/app/building-your-application/routing)) 支持[布局](/docs/app/building-your-application/routing/pages-and-layouts), [服务器组件](/docs/app/building-your-application/rendering/server-components), [流动](/docs/app/building-your-application/routing/loading-ui-and-streaming), 和[并置数据提取](/docs/app/building-your-application/data-fetching).

使用`app`目录将自动启用[React 严格模式](https://react.dev/reference/react/StrictMode).了解如何[逐步采用`app`](/docs/app/building-your-application/upgrading/app-router-migration#migrating-from-pages-to-app).
