[服务器组件](/docs/app/building-your-application/rendering/server-components)和[路由处理程序](/docs/app/building-your-application/routing/route-handlers)中使用的依赖项将自动由 Next.js 绑定。

如果依赖项使用 Node.js 特定的功能，则可以选择从服务器组件绑定中选择不使用特定的依赖项，并使用本机 Node.js 的`require`。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    serverComponentsExternalPackages: ["@acme/ui"],
  },
};

module.exports = nextConfig;
```
