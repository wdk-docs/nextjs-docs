Next.js 提供[**gzip**](https://tools.ietf.org/html/rfc6713#section-3)压缩来压缩渲染的内容和静态文件。通常，您会希望在像[nginx](https://www.nginx.com/)这样的 HTTP 代理上启用压缩, 以卸载`Node.js`进程的负载。

要禁用**compression**，请打开`next.config.js`并禁用`compress`配置:

```js filename="next.config.js"
module.exports = {
  compress: false,
};
```
