---

nav_title: typedRoutes

---

对[静态类型链接](/docs/app/building-your-application/configuring/typescript#statically-typed-links)的实验性支持. 此功能需要在项目中使用应用程序路由器和 TypeScript。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    typedRoutes: true,
  },
};

module.exports = nextConfig;
```
