在开发模式下运行 Next.js 时，您可以配置日志记录级别以及是否将完整 URL 记录到控制台。

目前，`logging`仅适用于使用`fetch`API 获取数据。它还不适用于 Next.js 中的其他日志。

```js filename="next.config.js"
module.exports = {
  logging: {
    fetches: {
      fullUrl: true,
    },
  },
};
```
