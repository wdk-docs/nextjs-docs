此页面提供了 Next.js 项目的文件和文件夹结构的概述。它涵盖了`app` 和 `pages`目录中的顶级文件和文件夹、配置文件以及路由约定。

## 顶级文件夹

|                                                                          |                        |
| ------------------------------------------------------------------------ | ---------------------- |
| [`app`](/docs/app/building-your-application/routing)                     | 应用程序路由器         |
| [`pages`](/docs/pages/building-your-application/routing)                 | 页面路由器             |
| [`public`](/docs/app/building-your-application/optimizing/static-assets) | 待服务的静态资产       |
| [`src`](/docs/app/building-your-application/configuring/src-directory)   | 可选的应用程序源文件夹 |
