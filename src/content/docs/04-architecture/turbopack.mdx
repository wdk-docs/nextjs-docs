[Turbopack](https://turbo.build/pack) (beta) 是一个针对 JavaScript 和 TypeScript 优化的增量 bundler，用 Rust 编写，并内置到 Next.js 中。

## 用法

Turbopack 可以在 Next.js 的`pages`和`app`目录中使用，以加快本地开发。
要启用 Turbopack，请在运行 Next.js 开发服务器时使用`--turbo`标志。

```json filename="package.json" highlight={3}
{
  "scripts": {
    "dev": "next dev --turbo",
    "build": "next build",
    "start": "next start",
    "lint": "next lint"
  }
}
```

## 支持的功能

要了解更多关于 Turbopack 当前支持的功能，请查看[文档](https://turbo.build/pack/docs/features).

## 不支持的功能

Turbopack 目前只支持`next-dev`，不支持`next-build`。随着我们向稳定迈进，我们目前正在努力为建筑提供支持。
